Assignment 7

 

Recap:

In the seventh session, we dealt with the Radio feature in the Micro:bit. To summarise:

  1. The radio feature allows one to communicate wirelessly between multiple Micro:bits. 
  2. The radio link can also be used to communicate with a Bluetooth device.
  3. We discussed the broadcast mode of communication where we have one transmitter and multiple receivers.
  4. The concept of channels in radio communication was explained.
  5. Channel settings in Micro:bits help multiple transmitters to transmit information for the receiver to tune in to any transmitter of its choice. 
  6. Before any communication, it is best to set the radio channel. A Micro:bit can only listen to one channel (Group ID) at any time.
  7. Simple radio project involving one transmitter and receiver was demonstrated.
  8. Coded programs, simulated the program, saved the programs on the laptop, downloaded the program in the Micro:bit and ran the program on the Micro:bit.
  9. Discussed the various stages of creating the complete project -- requirement, design, code, simulate, check, store, download (flash) and run.
  10. Advanced students were asked to check the Javascript code and modify its contents.

 

The class projects as described above can be completed and saved in the Desktop folder.

 

What did they learn:
Technology Related

1.       Radio transmit and receive unit in Micro:bit

2.       Radio communication requires an antenna for wireless communication

Science Related

1.       How does radio communication work?

2.       The difference between broadcast and unicast.

3.       The importance of channels (Group ID)

Coding Related

1.       radio set group to set up the radio channel.

2.       Transmit a number through radio send number command

3.       Transmit a string through radio send string command

4.       Receive radio number through on radio received receivedNumber command

5.       Receive radio string through on radio received receivedString command

Project A:
Daddy Stealing my Chocolate
The aim of the project is to design an electronic warning system which warns when the fridge is opened that will make sound and flashlight on the screen.

1.       On start

o    show a heart with a pause of 2sec to check if the download is successful.

o    set the radio group to 99 (any channel no. )

2.       On the transmitter side:

o     loop forever to read the light level.

o    If the light level is greater than 30 the radio send 150 (magic number) through radio send number command.

3.       On the receiver side:

o    Use on radio received number to read the transmitted data.

o    If the received number is equal to 150 then brighten up the screen and play a music.
