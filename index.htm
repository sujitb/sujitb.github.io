Recap:

In the second session, the Micro:bit buttons were used as a sensor. To summarise we:

  • Made use of the Buttons to display icons (images) and also played some music.
  • We know that there are 3-buttons. The Button-A and the Button-B can be programmed whereas the Reset button on the back of the Micro:bit cannot be programmed and can be used only for re-starting the program.
  • With 2-Buttons we can have 3-different ways of taking action -- on button A pressed, on button B pressed and on button A+B pressed.
  • The other 2-ways which is long-press button A and log-press button B will be used later on.
  • Coded programs, simulated the program, saved the programs on the laptop, downloaded the program in the Micro:bit and ran the program on the Micro:bit.
  • We now know the difference between a switch and a push-button. 
  • We used buttons as sensors, Micro:bit program as the processing power and the screen as an actuator. All three elements sensor, brain and actuator were in the Micro:bit. 
  • We have also connected a speaker through crocodile clips by connecting the Pin 0 and GND of the Micro:bit to a speaker jack.
  • We can also use the speaker without the crocodile clips by using kitchen Aluminium foil as shown in the figure below.
  • Discussed the various stages of creating the complete project -- requirement, design, code, simulate, check, store, download (flash) and run.
  • In the software platform Makecode, we can press the buttons on the screen with the mouse to simulate the program. For A+B action, we have seen an additional button A+B appearing the screen for us to trigger.

Advanced students were asked to check the Javascript code and modify its contents.

 

 

 

What did they learn:

Technology Related

  • What is the difference between a switch and a push-button
  • Push-button comes back to its original state as soon as the force is removed.
  • With 2-Buttons we can have 5-trigger events. 
  • Connect an external speaker. By default Pin 0 and GND is used to connect to the speaker through crocodile clips.
  • Reset Button cannot be programmed and used only to start the program from start.

Science Related

  • Both switch and push-button can be used to complete the electric circuits.

Coding Related

  • New functions
    • on button A pressed
    • on button B pressed
    • on button A+B pressed
  • Show icon
  • Play music

On button A/ B/ A+B is outside the commands on start and forever (loop)
