Beginners Assignment 01

Recap:

In the first session, we made excellent progress. To summarise we:

  Made use of the Light Emitting Diode (Red LEDs) to display string (text), icons (images), animated figures and numbers.

  Coded programs, simulated the program, saved the programs on the laptop, downloaded the program in the Micro:bit and ran the program on the Micro:bit.

  Know that LED has a positive (long leg) end and negative (short leg) end. It allows the current to flow in one direction.

  We also now know various components which are present in the front and back of the Micro:bit.

  Discussed the various stages of creating the complete project -- requirement, design, code, simulate, check, store, download (flash) and run.

  Discussed the difference between the human brain and the computer brain.

  We know various features of the Microsoft's visual software platform Makecode.

  Advanced students were asked to check the Javascript code and modify its contents.

 

What did they learn:
Technology Related

  What is a Light Emitting Diode (Red LEDs)

  How LED works 

  LEDs have anode (positive) and cathode (negative)

  LED allows the flow of current in one direction.

  Reset Button in Micro:bit to start the program from start.

Science Related

  Importance of Red light 

  Less scattering and hence visible from a far distance.

  1000ms = 1sec

Coding Related

  Show string

  Show icon

  Show number

  Pause for some time

  Clear Screen

  Difference between on start and forever (loop)

 

Projects completed in Class:

  Display any string (text), including scrolling text.

  Display icons (images), including how to create your own image

  Display a number, including countdown

  Display animated images
